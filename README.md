Devoir ou je devais
Réaliser un site web complètement fonctionnel en utilisant uniquement HTML et CSS, sans JavaScript ni langage back-end.
Vous êtes libres de créer n’importe quel type de site : vitrine, portfolio, galerie, page d’hommage, site sur un hobby, etc.

Contraintes HTML (fichier index.html)

1. Images obligatoires
   · Ajoutez au moins 3 images pertinentes par rapport au thème du site.
   · Chaque image doit posséder un attribut alt renseigné avec une description précise (accessibilité).
2. Interdiction d’utiliser abusivement les balises <div> et <span>
   · Privilégiez les balises sémantiques : <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, etc.
Contraintes CSS (fichier séparé, ex. style.css)

1. Liaison HTML/CSS
   · Le fichier CSS doit être relié au HTML via la balise <link> dans l’en-tête (<head>).
2. Police Google Fonts
   · Importez au moins une police depuis Google Fonts (ex. Roboto, Open Sans, Lato…).
   · Appliquez‑la à un ou plusieurs éléments de la page.
3. Mise en page
   · Utilisez Flexbox (display: flex) ou Grid Layout (display: grid) pour organiser votre contenu (par exemple, une galerie, une barre de navigation, une section en colonnes).
   · Vous pouvez combiner les deux.
4. Espacements
   · Utilisez les propriétés margin et padding pour aérer la page et rendre la navigation agréable.
5. Couleurs
   · Choisissez vos couleurs avec des codes hexadécimaux (#...), RGB (rgb(...)) ou RGBA (rgba(..., ..., ..., transparence)).
   · Évitez les noms de couleurs prédéfinis (red, blue, etc.) sauf en complément.
6. Dégradé
   · Utilisez la fonction linear-gradient() au moins une fois (fond d’un élément, d’un bouton, d’un header…).
